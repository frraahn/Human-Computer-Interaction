## 🎓 Course Overview
Human-Computer Interaction is a multidisciplinary field focused on the design, evaluation, and implementation of interactive computing systems for human use. The primary goal of this course is to transition from technology-centered design to **user-centered design**, ensuring software and hardware interfaces are intuitive, accessible, and efficient.

### Objectives:
* Understand human cognitive and physical limitations and how they impact design.
* Master industry-standard prototyping workflows (low-fidelity sketches to high-fidelity interactive wireframes).
* Conduct rigorous usability testing, heuristic evaluations, and user research.

## 💡 Course Reflection & Key Takeaways

The most important lesson from this course is that "you are not your user." 

Throughout these assignments, I learned to move past my own design assumptions by using actual data from user testing and heuristic evaluations. Seeing where real users stumbled on my prototypes completely changed how I think about interface layout and accessibility. True UX design is an iterative process of testing, learning, and refining.
